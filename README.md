# Mehmed Asım Deliser – Kişisel Web Sayfası

Web CBS Dersi – Ödev 1

## Dosyalar

| Dosya | Açıklama |
|---|---|
| `index.html` | Ana sayfa – Hakkımda, İş Deneyimi, Eğitim, Beceriler |
| `hobilerim.html` | Hobilerim + Leaflet haritası (kariyer & yaşam noktaları) |
| `iletisim.html` | İletişim bilgileri ve sertifikalar |

## Özellikler

- ✅ En az üç farklı başlık ve HTML dosyası (Hakkımda, Hobilerim, İletişim)
- ✅ Leaflet haritası (`hobilerim.html` içinde – iş, eğitim ve yaşam noktaları, katman filtreleri)
- ✅ CSS animasyonları (fadeSlideUp, pulse, orb-float, blink, scroll-triggered reveals)
- ✅ GitHub Pages uyumlu (tüm bağımlılıklar CDN üzerinden)

## GitHub Pages Yayınlama

1. Bu klasörün içeriğini bir GitHub reposuna push edin.
2. Repo ayarlarından **Pages → Branch: main / root** seçin.
3. Birkaç dakika sonra `https://<kullanıcıadı>.github.io/<repo-adı>/` adresinden erişilebilir.

## Teknolojiler

- Saf HTML5 / CSS3 / Vanilla JS
- [Leaflet.js](https://leafletjs.com/) v1.9.4
- [CartoDB Dark tiles](https://carto.com/basemaps/) (OpenStreetMap verileri)
- Google Fonts: Playfair Display + DM Sans
